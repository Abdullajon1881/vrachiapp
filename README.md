# 🏥 Vrachiapp - Медицинская платформа

Современная веб-платформа для связи пациентов с врачами, управления медицинскими консультациями и административного контроля медицинских услуг.

## ✨ Основные возможности

### 👥 Для пациентов
- **Регистрация и авторизация** с поддержкой Google OAuth
- **Поиск врачей** по специализации и местоположению
- **Запись на консультации** к выбранным специалистам
- **Личный кабинет** с историей консультаций
- **Профиль пациента** с медицинской информацией

### 👨‍⚕️ Для врачей
- **Подача заявки** на получение роли врача
- **Профессиональный профиль** с информацией о специализации
- **Управление консультациями** и расписанием
- **Загрузка документов** (диплом, лицензия, фото)
- **Многоязычная поддержка** консультаций

### 🔧 Для администраторов
- **Полное управление пользователями** (пациенты, врачи, администраторы)
- **Модерация заявок врачей** (одобрение/отклонение)
- **Редактирование профилей** всех пользователей
- **Изменение ролей** пользователей
- **Удаление пользователей** из системы

## 🛠 Технологический стек

### Frontend
- **React 18** - современная библиотека для создания пользовательских интерфейсов
- **SCSS** - препроцессор для стилизации с поддержкой переменных и миксинов
- **Vite** - быстрый сборщик проекта
- **Responsive Design** - адаптивный дизайн для всех устройств

### Backend
- **Django 4.2** - мощный веб-фреймворк на Python
- **Django REST Framework** - API для взаимодействия с фронтендом
- **MySQL** - надежная реляционная база данных
- **Django ORM** - объектно-реляционное отображение

### Аутентификация
- **JWT токены** - безопасная аутентификация
- **Google OAuth** - вход через Google аккаунт
- **CSRF защита** - защита от межсайтовых запросов

## 🚀 Быстрый старт

### Предварительные требования
- Python 3.12+
- Node.js 18+
- MySQL 8.0+
- Git

### Установка и настройка

1. **Клонирование репозитория**
```bash
git clone https://github.com/your-username/vrachiapp.git
cd vrachiapp
```

2. **Настройка Backend**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

pip install -r requirements.txt
```

3. **Настройка базы данных**
```bash
# Создайте базу данных MySQL
mysql -u root -p
CREATE DATABASE vrachiapp_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# Примените миграции
python manage.py migrate

# Создайте суперпользователя
python manage.py createsuperuser
```

4. **Настройка Frontend**
```bash
cd ../src
npm install
```

5. **Запуск приложения**
```bash
# Backend (в папке backend)
python manage.py runserver

# Frontend (в папке src)
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## 🔐 Роли пользователей

### Пациент (patient)
- Базовый доступ к системе
- Поиск и запись к врачам
- Управление личным профилем

### Врач (doctor)
- Все возможности пациента
- Профессиональный профиль
- Управление консультациями
- Загрузка документов

### Администратор (admin)
- Полный доступ к системе
- Управление всеми пользователями
- Модерация заявок врачей
- Системные настройки

## 📊 Функции администратора

### Управление пользователями
- Просмотр всех пользователей системы
- Редактирование профилей (имя, фамилия, роль, контакты)
- Изменение ролей (пациент ↔ врач ↔ администратор)
- Удаление пользователей из системы

### Модерация заявок врачей
- Просмотр всех заявок на роль врача
- Одобрение/отклонение заявок с указанием причины
- Автоматическое обновление профиля при одобрении
- Просмотр загруженных документов (диплом, лицензия, фото)
