# 🏥 Healzy - Инновационная медицинская платформа

Современный веб-сайт медицинского центра с адаптивным дизайном, темной темой и интуитивным интерфейсом.

## ✨ Основные возможности

### 🎨 Дизайн и интерфейс
- **Адаптивный дизайн** - отлично выглядит на всех устройствах
- **Темная/светлая тема** - переключение одним кликом
- **Современный UI** - градиенты, тени, плавные анимации
- **Медицинская тематика** - адаптировано под медицинские услуги

### 📱 Компоненты

#### 🏠 Главная страница (Hero)
- **Привлекательный заголовок** с описанием услуг
- **Статистика** - количество пациентов, врачей, лет опыта
- **Призыв к действию** - кнопки для записи и консультации

#### 🛠️ Услуги (Services)
- **6 основных услуг** с иконками и описанием
- **Мобильная карусель** - горизонтальный скролл на мобильных
- **Карточки услуг** с категориями и кнопками "Подробнее"
- **Адаптивная сетка** - 3 колонки на десктопе, карусель на мобильных

#### 🧭 Навигация
- **Боковая панель** - иконки с подсказками при наведении
- **Мобильная навигация** - нижняя панель на мобильных
- **Переключатель темы** - иконка луны/солнца в сайдбаре

#### 📞 Контакты и информация
- **Header** - телефон, язык, пользователь, поддержка
- **Footer** - социальные сети, разделы, контакты

## 🎯 Функциональность

- **SCSS** - препроцессор стилей

### Backend
- **Django 4.2** - веб-фреймворк
- **Django REST Framework** - API
- **MariaDB/MySQL** - база данных
- **JWT** - аутентификация
- **Google OAuth** - социальная аутентификация

### Стилизация
- **SCSS переменные** - централизованные цвета и размеры
- **CSS Grid** - адаптивные сетки
- **Flexbox** - гибкая верстка
- **CSS Transitions** - плавные анимации

### Адаптивность
- **Mobile First** подход
- **Breakpoints**: 480px, 768px, 1024px
- **Touch-friendly** интерфейс

## 🚀 Установка и запуск

### Frontend

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

### Backend

```bash
# Переход в папку backend
cd backend

# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Создание базы данных MariaDB/MySQL
# CREATE DATABASE vrachiapp_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# Применение миграций
python manage.py makemigrations
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
```

**Frontend**: http://localhost:5173  
**Backend API**: http://localhost:8000  
**Django Admin**: http://localhost:8000/admin/

## 🔧 Настройка

### Переменные SCSS
Все цвета, размеры и другие параметры можно изменить в `src/styles/variables.scss`:

```scss
// Цвета
$primary-blue: #2563eb;
$light-blue: #dbeafe;

// Размеры
$border-radius-lg: 16px;
$spacing-lg: 24px;
```

### Темная тема
Добавление новых компонентов в темную тему:

```scss
.component {
  // Светлая тема
  background: $white;
  
  // Темная тема
  .dark-theme & {
    background: $dark-bg;
  }
}
```
